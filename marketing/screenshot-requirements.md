## Screenshot Standards

### Technical Requirements
- **Format:** PNG (no compression artifacts)
- **Resolution:** 1280x800 minimum (2560x1600 retina preferred)  
- **File size:** Under 500KB each (optimize for directory uploads)
- **Browser:** Chrome/Safari, no extensions, clean session
- **Shadows:** Subtle browser chrome/window shadow for depth

### Visual Guidelines
- **No browser UI** unless specifically relevant (for integration screenshots)
- **Clean data** - No Lorem Ipsum, use realistic sample data
- **Consistent branding** - Same logo/colors across all shots
