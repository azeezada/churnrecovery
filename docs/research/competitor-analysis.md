## 1. Churnkey

**URL:** https://churnkey.co
**Founded:** ~2021
**Type:** Full-stack churn reduction SaaS (PRIMARY COMPETITOR)

### Core Features
- **Cancel Flows** — Embeddable, personalized cancellation interstitials with offers (discounts, pauses, plan changes, trial extensions, CS interventions). Claims up to 58% reduction in voluntary cancellations.
- **Payment Recovery / Dunning** — ML-powered retry logic, personalized multi-channel dunning emails, Pause Wall, Failed Payment Wall. Claims up to 89% recovery.
- **Reactivation Campaigns** — Win-back sequences targeting churned customers via segmentation.
- **Insights AI / Feedback AI** — AI-powered analysis of cancellation feedback; auto-sorts and ranks reasons.
- **Customer Timelines** — Per-account audit trail of lifecycle events.
- **Account Agent** — AI agent that manages retention actions autonomously (Intelligence tier only).
- **A/B Testing** — For cancel flows, dunning copy, and offer structures.
- **Adaptive Offers** — AI-powered offer optimization (Intelligence tier).
- **Compliance Automation** — Automated handling of regulatory cancellation requirements.
- **Self-Improving Precision Retries** — ML-optimized payment retry timing.
- **AI-Powered Translations** — Auto-translates flows into multiple languages.
- **Managed Services** — Done-for-you retention option.
- **Stripe App** — Named Stripe App of the Month.
- **Cooldown periods** — Prevents customers from gaming discount offers repeatedly.

### Pricing
| Plan | Price | Details |
|------|-------|---------|
| Starter | $250/mo yearly | <$5k/mo churn. Cancel Flows, Payment Recovery, Customer Timelines |
| Core | $700/mo yearly | $20k churn volume. A/B Testing, Segmentation, Rules-based Retry, Slack + Webhooks |
| Intelligence | $825/mo yearly | $20k churn volume. Adaptive Offers, Account Agent, Feedback AI, AI translations, Self-Improving Retries |

### Target Market
- Subscription SaaS of all sizes, particularly B2B
- Smallest viable: <$5k/mo churn volume

### Differentiators
- Full-stack (voluntary + involuntary + reactivation)
- AI-first with depth (Feedback AI, Adaptive Offers, Account Agent)
- Fast integration (35 min claimed)
- White-labeled
- Stripe App of the Month credibility

### Weaknesses
- Pricing opaque at scale (churn volume-based)
- Limited public case studies
- No native CRM integrations (vs ProsperStack's HubSpot/Salesforce)
- Narrow integration list

## 2. Raaft

**URL:** https://raaft.io
**Type:** Lightweight cancellation flow SaaS

### Core Features
- No-code cancellation flow builder
- Exit surveys with predefined motivators + custom text
- Real-time analytics dashboard
- Cancel Request Mode (routes to CS instead of auto-cancel)
- Conditional offers based on cancel reason

### Pricing
- **$20/month** with 30-day free trial (no credit card to start)
- Paid plans not publicly disclosed
- Claims 20% retention rate, 7x ROI

### Target Market
- Early-stage to SMB SaaS
- Budget-conscious teams
- 100+ customers

### Differentiators
- Only major competitor with permanent free plan
- Simplest setup (30 min)
- 5/5 on G2

### Weaknesses
- No dynamic personalization
- Limited design customization
- **No payment recovery / dunning**
- **No reactivation campaigns**
- Small customer base

## 3. Chargebee Retention (formerly Brightback)

**URL:** https://chargebee.com/retention/
**Acquired:** 2022 by Chargebee
**Type:** Cancel experience + retention automation (enterprise-focused)

### Core Features
- Personalized cancel pages with segment-based design/messaging/offers
- Deflection funnel (canceled / deflected / watch list / saved)
- Three targeting types: randomized, rules-based, ML "Smart Targeting"
- A/B testing at scale (hundreds-thousands of tests/day)
- Customer profiling for high-value accounts
- Retained revenue reporting
- Workflow triggers (Slack/email alerts)
- Segment CDP integration
- No-code experience manager
- Win-back campaigns

### Pricing
- 30-day free trial, contact sales for pricing
- Enterprise-oriented pricing (expensive)

### Target Market
- Mid-market to enterprise
- Best fit for existing Chargebee billing customers

### Differentiators
- Native Chargebee ecosystem integration
- ML-based Smart Targeting
- Beautiful dashboards
- High-volume testing capability
- Segment native integration

### Weaknesses
- **Locked into Chargebee ecosystem**
- Slow implementation (months)
- Rigid pricing/paywalls, stifling contracts
- **No payment recovery/dunning** — voluntary churn only
- Expensive, poor fit for SMBs

|------|-------|-------------|
| Grow | From $200/mo | 50–500 |
| Prosper | From $750/mo | 500–50,000 |
| Enterprise | Custom | Unlimited |

- Subscription businesses of all sizes

**Type:** Subscription analytics platform with recovery add-ons

### Core Features
- **Core product:** MRR dashboard, churn tracking, forecasting, segmentation
- **Cancel Insights:** Cancellation surveys + analytics
- **Recover:** Failed payment dunning (email campaigns + in-app prompts)
- Benchmarking against anonymous industry data
- Augmentation (enrich customer data)

### Pricing
- Analytics: from $108/mo
- Recover: from $58/mo
- Cancel Insights: included in higher tiers

### Target Market
- SaaS companies needing analytics first, retention second
- Stripe-first companies

### Differentiators
- Best-in-class subscription analytics
- Industry benchmarking
- Low entry price for dunning ($58/mo)

### Weaknesses
- **Cancel flows are basic** (surveys only, no interactive offers)
- Dunning is basic (email-based, no ML retries)
- Analytics-first company — retention is a secondary product
- No AI features

## Market Gap Analysis

### Where ChurnRecovery Can Win

1. **Price undercut with full stack** — Churnkey starts at $250/mo. A $99/mo or even free-tier entry with cancel flows + basic dunning would capture the SMB market that Raaft targets but can't serve fully.

2. **Developer-first experience** — Most competitors have mediocre developer docs. A best-in-class SDK, open-source widget, and great API docs would differentiate.

3. **Transparent pricing** — Almost everyone in this space has opaque pricing. Flat, predictable pricing builds trust.

4. **Open-source cancel flow widget** — No one offers this. An open-source embeddable widget (like Stripe Elements) with a paid backend for analytics/offers would be a massive GTM wedge.

5. **AI-native from day 1** — Churnkey bolted on AI. Building with AI at the core (adaptive offers, feedback synthesis, churn prediction) from the start means better architecture.

6. **Modern stack / fast integration** — Target <15 minute integration time. Code snippet + Stripe API key = done.

7. **Self-serve onboarding** — Churnkey and ProsperStack push demo calls. Full self-serve with optional upgrade to human support.

8. **Billing platform agnostic** — Start with Stripe (80% of market), add Chargebee and Paddle support quickly. Don't lock into one ecosystem.
