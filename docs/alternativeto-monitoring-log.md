# AlternativeTo Monitoring Log

## How to Update

Check your listing weekly: search "ChurnRecovery" at https://alternativeto.net

**What to track:**
- **Likes/upvotes** — your listing's vote count (visible on the listing page)
- **Reviews** — number of written reviews
- **Traffic (est.)** — check Cloudflare Analytics → Referrers → `alternativeto.net` for weekly session count
- **Actions taken** — anything you did that week (responded to a comment, asked users to upvote, added to a competitor page, etc.)

Update this log every Monday. Takes ~5 minutes.
